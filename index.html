Welcome to My Internship Presentation

This page outlines the key topics and technologies I explored during my internship.

Introduction

I started my internship with a curiosity about IoT and automation.

Node-RED

I used Node-RED to create simple automation flows.

Node-RED is a flow-based development tool for visual programming. I used it to create logic for smart home components like motion detectors and smart lights.

Side Project

Developed a small smart home system using sensors and relays.

The project included temperature monitoring, motion detection, and remote control of devices. I integrated it with a simple dashboard for real-time updates.

Challenges & Learnings

Faced issues with device integration and data flow consistency.

I learned to debug MQTT data issues and optimize flow timing. Working with asynchronous data taught me the value of real-time debugging tools.

Conclusion

This internship was a great step forward in understanding IoT platforms and flow-based logic.
